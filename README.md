# Matte 商城

一个基于 Java + SQLite 的轻量级电商系统。

## 技术栈

- Java 17
- Maven
- SQLite
- 内置 HttpServer（无需 Tomcat）

## 启动方式

默认访问地址：`http://localhost:8080`

## 功能介绍

### 用户端

| 功能 | 说明 |
|------|------|
| 用户注册/登录 | 支持普通用户注册，密码使用 SHA-256 加密存储 |
| 账号安全 | 登录失败 10 次锁定 1 小时，50 次自动禁用，管理员可解锁 |
| 商品浏览 | 按分区展示商品，支持查看价格、库存、图片 |
| 购物车 | 加入购物车、修改数量、自定义单价、结算 |
| 地址管理 | 添加/编辑/删除收货地址，支持设置默认地址 |
| 订单结算 | 选择商家配送，支持选择收货地址 |

### 管理员端

访问 `/admin.html` 进入管理后台。

| 功能 | 说明 |
|------|------|
| 商品管理 | 添加、编辑、删除商品，上传商品图片 |
| 分区管理 | 添加、编辑、删除商品分区/分类 |
| 订单管理 | 查看所有订单，标记发货、确认送达 |
| 用户管理 | 查看用户列表，解锁账号、解除禁用 |
| 商家管理 | 添加配送商家，上传收款二维码 |
| 页面设置 | 设置商城标题、标语、首页背景图、付款二维码 |

### 控制台命令

启动后可在控制台输入以下命令：

| 命令 | 说明 |
|------|------|
| `users` | 查看所有用户 |
| `set-admin <用户名>` | 将指定用户设为管理员 |
| `help` | 查看帮助 |
